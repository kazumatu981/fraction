## モチベーション

## 演習概要

```ts
import { Fraction } from 'fraction';

// 1/2 を表すオブジェクト
const f1 = new Fraction(1, 2);
// 1/3 を表すオブジェクト
const f2 = new Fraction(1, 3);

// 1/2 + 1/3 を計算する
const result = f1.add(f2);

console.log(result.toString()); // 5/6
```
